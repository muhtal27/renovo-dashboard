Installation

How to install it

Gmail

  1. Select the signature block in the preview above (click and drag from "Muhammad" through the bottom line).
  2. Copy (Cmd+C / Ctrl+C).
  3. In Gmail: Settings → See all settings → General → Signature. Create a new signature, paste into the editor, save at the bottom.

Apple Mail

  1. Open this page in Safari, select and copy the signature block.
  2. In Apple Mail: Mail → Settings → Signatures. Create a new signature, paste in, uncheck "Always match my default message font" so the formatting survives.

Outlook (Mac / Web)

  1. Select and copy the signature block above.
  2. In Outlook web: Settings → Mail → Compose and reply → Email signature. Paste and save.
  3. In Outlook for Mac: Outlook → Settings → Signatures. Same process.
