# Ayesha Ali Khan — Brand Identity

Logo system for **Ayesha Ali Khan**, an elite-class makeup studio. Standalone
project kept inside this repo for convenience; nothing is wired into the
Next.js app.

## Concept

A quiet, couture mark. An **AAK** monogram set in a high-contrast Didone serif
sits inside a double-ring medallion, framed by hairline flourishes. The center
`A` is taller, creating a subtle crown axis. A diamond-rule separates the
wordmark from the `MAKEUP STUDIO` tagline and an `ELITE CLASS ATELIER` accent.

## Files

| File | Use |
| --- | --- |
| `logo-primary.svg` | Full vertical lockup on obsidian ink |
| `logo-light.svg` | Full vertical lockup on cream (print / stationery) |
| `logo-horizontal.svg` | Header, signage, email signature |
| `logo-monogram.svg` | Social avatar, favicon, embossed details |
| `index.html` | Brand preview — open in a browser |

## Palette

| Token | Hex | Role |
| --- | --- | --- |
| Obsidian Ink | `#0D0A08` | Primary background |
| Atelier Gold | gradient `#D9A66A → #F3D7A5 → #B98447 → #8A5A2B` | Mark & ornament |
| Antique Brass | `#B98447` | Accent rules, small caps |
| Champagne Cream | `#F6EFE3` | Light background |
| Pearl | `#F7ECD7` | Wordmark on dark |

## Typography

- **Display / Wordmark:** Didot → Bodoni MT → Playfair Display (serif fallback)
- **Tagline / Accents:** Montserrat, tracked wide (300–500)

## Preview

Open `index.html` in any browser — no build step, the SVGs render inline from
the same folder.
